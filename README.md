# beacon_flood
가짜 비콘 프레임을 생성하여 broadcast로 전송해서 가짜 Wi-Fi를 생성한다. <br>
스마트폰의 Wi-Fi 목록을 확인하면 ssid 목록파일에 있는 ssid가 나타나는 것을 볼 수 있다. <br><br>
사용방법 : [프로그램 경로] [인터페이스 이름] [ssid 목록파일 경로] <br>
ex) ./beacon_flood wlan0 ssid-list.txt <br><br>
